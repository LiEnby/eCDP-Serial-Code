Create JAR and run application:
-------------------------------
1. Install at least Java 11 JDK (https://adoptopenjdk.net/)
2. Install Maven (https://maven.apache.org/download.cgi)
3. mvn clean package
4. Run 'eCDPPasswordGeneratorApp_<version>.jar' to start password generator.
   For instance via console: java -jar eCDPPasswordGeneratorApp_<version>.jar
